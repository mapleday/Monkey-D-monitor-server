access_keyword_list 保存的是64期敏感词的和敏感表情图，过滤的规则就是只要url内包含或者post body就会返回300009错误

comment_switch 是一个开关的zset里面保存的是各种开关，目前只有一个开关access_key_word 控制是否过滤url敏感词， 关闭的方式就是从zset中删除此value

mobile_ban_list 是手机号黑名单，凡是被禁言的用户，在大V下发言会把绑定的手机号记录在此zset中。 再次黑名单中的手机号所绑定的psssport是无法发言的

passport_ban_list 是用户passport黑名单，在此名单中的passportId无法使用sns的任何接口。

vip_list 大v的列表，在此列表中的大v，针对其feed转发或者评论会触发手机号黑名单的过滤。
